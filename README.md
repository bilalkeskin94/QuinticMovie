## Setup Direction

npm install or
yarn install.

npm run dev or
yarn dev -o

## Enviroment Creation

Create a .env file

NUXT_ENV_API_KEY = "API_KEY"
BASE_URL = "https://the-one-api.dev/v2/"

You can find the api key from https://the-one-api.dev/
